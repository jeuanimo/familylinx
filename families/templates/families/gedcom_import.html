What is GEDCOM?

GEDCOM is the standard file format used by genealogy software. Export from:

  • Ancestry.com – Go to Trees → Export Tree
  • FamilySearch – Download GEDCOM from any tree
  • MyHeritage – Export from Family Tree Builder
